# One Take Wonder Marketing Site

This repository now contains a static marketing website for **One Take Wonder Marketing**, ready to publish with **GitHub Pages**.

## Files
- `index.html` - homepage content and structure
- `styles.css` - custom styling and responsive layout
- `Logo/Logo.png` - agency logo
- `MarketingBanner.jpg` - hero banner image

## Publish on GitHub Pages
1. Push this repo to GitHub.
2. In GitHub, open **Settings > Pages**.
3. Under **Build and deployment**, set:
   - **Source:** Deploy from a branch
   - **Branch:** `main` (or your default branch) and `/ (root)`
4. Save. Your site will be published at the GitHub Pages URL shown there.

## Optional custom domain
If you want a branded domain later, add a `CNAME` file in the repo root and set DNS records with your domain provider.
